# avaks_checklists

Приложения для создания чек-листов с помощью шаблонов и сохранение результатов 

## Пример шаблона
Пример: [example.md](example.md)
## Техническая составляющая
В приложение импортируется .md файл-шаблон , при открытии шаблона начинается заполнение чек-листа, как работа с ним будет завершена, нажимайте на иконку «сохранить» в панели справа, открывается file-picker куда сохранить итоговый zip-архив. Zip-архив содержит txt файл записи выполненых и не выполненых задач, а так же файлы, прикреплённые в определённые места файлы. 
## Проект изнутри
Каталог Services — Функции, которые работают для работы с файловой системой или парсом данных из .md файлов
Каталог Screens — Все экраны приложения, которые существуют в нём
Каталог Models — Заготовки для типов данных, которые (после импортируются в функции, например, для обработки отделения [checkbox] от, например, [voice])
main.dart — запуск самого приложения
